# Super Textbox

A Budibase custom component that exposes the styled copyable `Textbox` from `@poirazis/supercomponents-shared`.

## Features
- Displays a value in a clean monospace bordered box
- One-click copy to clipboard with visual feedback ("Copied !")
- Hover reveals the copy icon
- Supports bindings for the value
- Respects Budibase styling (padding, size, border, background)

## Usage
1. Add the **Super Textbox** component to your screen.
2. Bind or enter a value in the **Value** setting.
3. Optionally disable it or customize the copy messages.

The component is a thin wrapper around the shared library Textbox for consistent styling across your Budibase apps.
